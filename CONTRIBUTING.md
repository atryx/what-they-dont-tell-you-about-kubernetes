# Contributing

Thanks for wanting to contribute! This repo collects **real-world Kubernetes gotchas** — the pain points that official docs gloss over.

## How to Contribute

1. Fork this repo
2. Add your gotcha under the appropriate section (or create a new one)
3. Follow the existing format:
   - **Clear title** describing the gotcha
   - **What happens** — the symptom or surprise
   - **Why it happens** — root cause
   - **How to fix / avoid it** — actionable solution
   - **Real-world impact** — optional but appreciated
4. Submit a PR

## Guidelines

- Focus on **production pain points**, not beginner mistakes
- Include specific versions/contexts where relevant
- Be brutally honest but constructive
- No vendor bashing — focus on technical reality
- One gotcha per section, keep it scannable

## What Makes a Good Gotcha

✅ Something that cost you hours/days in production  
✅ A default that silently causes problems  
✅ A feature that works differently than documented  
✅ A cost or complexity that surprised your team  

❌ Basic misunderstandings of K8s concepts  
❌ Opinions without technical backing  
❌ Issues fixed in recent versions (unless noted)
